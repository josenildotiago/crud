# WIP

Um simples criador de crud para a GETIC

## Installação

```bash
composer require josenildotiago/crud
```

## Uso

Aṕos baixar o pacote, só seguir esse passo a passo

```bash
php artisan getic:install
```

## Passo a passo

- Será solicitado o nome da tabela.
- Escolha o template.
